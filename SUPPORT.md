# Support

## Getting Help

- Open an issue using one of the repository forms:
  - Scale Request
  - Idea
  - Package Expansion Request
  - Bug Report
- Use Discussions for open-ended questions and brainstorming.

## Security Issues

Do not file public issues for vulnerabilities. Use private security advisories.

## Triage Expectations

- New issues are auto-labeled with intake status.
- Weekly digest automation summarizes open demand and recent intake.
